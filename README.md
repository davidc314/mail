Pour lancer l'application double cliquer sur le dossier .app et configurer ensuite votre compte via l'interface prévue.

Une fois la configuration effectuée ferme l'interface pour sauvegarder la config et redémarrer l'application. 

Pour visualiser le contenu du projet, il suffit d'ouvrir le fichier .xcodeproj dans Xcode. Ainsi vous pourrez librement visualiser les commentaires non-présents dans l'AppleDoc. 

Veiller à ne pas configurer le client mail avec votre compte mail principale, je décline toute responsabilité en cas de perte de message.
